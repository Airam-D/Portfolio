Perfume NOCTIS

HTML5 CSS3

Estava querendo aprofundar meu conhecimento em CSS, então pensei em criar uma lading page de um perfume chamado NOCTIS, um perfume masculino ótimo para sair de noites com os amigos ou para encontros românticos. A página é simples, mas elegante, com um design moderno e responsivo que se adapta a diferentes tamanhos de tela. Utilizei técnicas avançadas de CSS para criar efeitos visuais atraentes, como transições suaves e animações sutis. O objetivo deste projeto foi aprimorar minhas habilidades em CSS e criar uma página visualmente atraente que destaque o produto de forma eficaz.
